# Contributor Covenant Code of Conduct

## Our Pledge
We as members pledge to make participation respectful and inclusive.

## Our Standards
Examples of behavior that contributes to a positive environment:
- Respectful communication
- Accepting feedback

## Enforcement Responsibilities
Project maintainers are responsible for clarifying and enforcing standards.

## Scope
This Code applies within all project spaces.

## Enforcement
Instances of abusive behavior may be reported to project maintainers.

## Enforcement Guidelines
Maintainers will follow guidelines in determining consequences.

## Attribution
This Code of Conduct is adapted from the Contributor Covenant.
